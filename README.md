### vote0.circ

简单投票表决器。

五个人投票。输入5个数，1表示通过，0表示否决。

### exam3.circ

输入一个8位0/1串（abcdefgh）和操作op。

根据op重排字符串。

若 op = 2，输出 acegbdfh；若 op = 4，输出 aebfcgdh。

### mips1.asm

写一个快排汇编代码。

先输入 n，表示要排序的数的个数。

然后输入 n 行，要排序的数。

按照格式输出排序后结果。

输入样例：

```
8
3
5
1
7
4
8
2
6
```

输出样例：

```
[1,2,3,4,5,6,7,8]
```
