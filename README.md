### mips2.asm

字符串逆置。

输入：n（表示字符串长度），x y（分别表示逆序的起止位置，从 0 开始计数），字符串 str

输出：讲这个字符串的[x,y]部分逆序后的结果。

输入样例：

```
10
2
5
qwertyuio
```

输出样例：

```
qwytreuio
```

输入圆盘个数 n

输入样例：

```
4
```
